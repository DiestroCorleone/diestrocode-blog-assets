Esto es lo que estaremos construyendo: Responsive Navbar Demo.
En resumen, es una barra de navegación que, llegado cierto breakpoint/ancho de pantalla, ha de colapsarse de forma vertical en lo que se llama comunmente un 'menú hamburguesa'. Y sin JavaScript ni frameworks, solo unas pocas líneas de CSS. Empecemos!


El HTML

Creamos un nav básico, lo único a tener en cuenta es que el mismo debe tener un id="menu"; el primer elemento a debe tener un href que apunte a ese id; y también verán que el texto un código Unicode: el correspondiente a las tres líneas horizontales que se ven cuando el menú se comprime y que se verá en pantallas pequeñas.
El segundo li será la X para cerrar el menú, y debe tener un href="#". El resto, serán los links a diversas partes de nuestro sitio.



El CSS

Comienzo agregando unos estilos básicos, como remover margin y padding del body, y darle un ancho y borde a nuestro nav para que se vea correctamente.



Aquí comemnzamos con lo importante: le damos un display flex y un justify-content: space-between a la lista desordenada dentro del nav, para que acomode nuestros links separados equitativamente. Y removemos el estilo a la lista para no tener los markers.
Luego, le indicamos a nuestro CSS que no muestre a los dos primeros elementos de la lista (el botón hamburguesa y el botón de cerrar), ya habrá oportunidad de hacerlos visibles en viewports pequeños. Además, agregué un poco de padding y cursor puntero para que el mouse nos muestre 'la manito' cuando nos apoyemos sobre ellos.



Media query

Para este ejemplo, elegí que la navbar se colapse a los 800px, así que en pantallas de este ancho o menores es cuando se activará la magia, como comento en el snippet.

  • Primero, la dirección del flex pasa a ser columna, para que se vea vertical.
  • Después, le decimos que se muestre el primer elemento li de nuestro nav que no sea target. Esto es lo que irá mostrando/ocultando el botón hamburguesa y la 'X' de cerrar según corresponda.
  • Después ocultamos los li que no sean target.
  • Y por último escondemos el primer li que sí lo sea, lo que trabaja en conjunto con el segundo punto para intercalar entre botón de menú/botón de cerrar.
